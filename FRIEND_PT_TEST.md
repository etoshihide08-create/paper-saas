# 友達PTテスト用の確認項目

## 目的
- RehaEvidence が「英語論文を日本語で理解し、保存・メモで学びを積み上げる」流れとして自然に使えるかを確認する
- どこで迷うか、どこで価値を感じるか、どのタイミングで Pro にしたくなるかを把握する

## テスト前に見てほしいこと
- 普段、英語論文をどのくらい読むか
- 臨床で使いたい論文を探すときに今いちばん困っていること
- 保存やメモをどのくらい日常で使うか

## テストで見てほしい流れ
1. 検索トップから気になるテーマで論文を探す
2. 検索一覧で「読みたい論文を選びやすいか」を見る
3. 論文詳細で日本語要約が理解しやすいか確認する
4. 気になった論文を保存する
5. 保存した論文をあとで探し直せるか確認する
6. 論文メモかクイックメモを書いてみる
7. 学ぶのおすすめ論文や掲示板を見てみる
8. マイページやプランを見て、課金したくなるか考える

## 聞きたいこと

### 1. 検索
- 検索結果は「これを読みたい」と選びやすいですか？
- 日本語タイトルや要約は十分でしたか？
- 違う論文が混ざっている感じはありませんでしたか？

### 2. 論文詳細
- 日本語要約は理解しやすいですか？
- 臨床参考度は納得感がありますか？
- 詳細ページは見やすいですか？ 情報量は多すぎませんか？

### 3. 保存
- 保存の流れは自然ですか？
- フォルダ分けはわかりやすいですか？
- 後から保存論文を探しやすいですか？

### 4. メモ
- メモは書きやすいですか？
- 論文を見ながらメモできる体験は役立ちますか？
- 引用やマーカーは使いたくなりますか？

### 5. 学ぶ
- おすすめ論文は見たくなりますか？
- 学習掲示板の投稿は自然ですか？
- 読んでいて「次も見たい」と思えますか？

### 6. 課金
- 無料でも十分使えますか？
- どこで Pro にしたくなりましたか？
- Pro に月額 1,500 円を払うなら、何が決め手になりますか？
- Expert が必要そうなのはどんな人だと思いますか？

## 最後に一言で聞きたいこと
- このサービスを一言でいうと何ですか？
- いちばん良かったところはどこですか？
- いちばん直してほしいところはどこですか？
- 友達の PT / OT に勧めたいですか？

## こちらで見たい数値
- 最初に検索するまでに迷ったか
- 1本目を読むまでに何回スクロールしたか
- 保存したか
- メモを書いたか
- プランページを見たか
- Pro にしたくなった瞬間があったか
